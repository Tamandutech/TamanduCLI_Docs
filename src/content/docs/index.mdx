## Repositórios

- **[TamanduCLI](https://github.com/Tamandutech/TamanduCLI)** — aplicativo desktop de linha de comando (Python) para conectar, enviar comandos e interagir com dispositivos via Bluetooth.
- **[TamanduCLI_DeviceCpp](https://github.com/Tamandutech/TamanduCLI_DeviceCpp)** — biblioteca C++ para integrar firmware com o protocolo wire do TamanduCLI.
- **[TamanduCLI_Docs](https://github.com/Tamandutech/TamanduCLI_Docs)** — código-fonte desta documentação, publicada em [tamandutech.github.io/TamanduCLI_Docs](https://tamandutech.github.io/TamanduCLI_Docs/).
